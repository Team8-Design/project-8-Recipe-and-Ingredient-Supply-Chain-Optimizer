# Project 8: Recipe & Ingredient Supply Chain Optimizer
大型餐饮企业食谱与食材供应链优化系统
基于四大核心数据结构：BST、单向链表、加权有向图、最大堆，分三大模块实现完整供应链业务。

## 核心功能
### Part1 基础工具箱
1. 实体建模：食材/食谱/供应商枚举类
2. BST食谱目录：字母序检索、菜系筛选、区间搜索
3. 单向链表：食谱食材动态增删、成本/库存校验
4. 加权有向供应图：供应商-食材采购网络
5. 最大堆采购优先级队列（自定义紧急度打分）
6. CSV数据集自动生成与加载（满足最低数据量要求）
7. 单元测试验证全部基础结构正确性

### Part2 高级算法与可视化
1. 食材替换无向图，按替换质量分排序推荐替代食材
2. 带可靠性阈值约束的最小成本供应商组合求解
3. 周菜单库存缺口分析、优先级采购计划生成
4. 多规模数据集性能基准测试+折线图可视化
5. Tkinter GUI可视化：食谱树、食材清单、供应网络、采购队列

### Part3 供应链危机恢复（开放挑战）
1. 核心供应商断供后风险食谱识别
2. 标记无法通过食材替换挽救的致命食谱
3. 最低成本、最高覆盖度替代供应商恢复方案规划

## 运行方式
1. 安装依赖：`pip install -r requirements.txt`
2. 执行入口脚本：`python main.py`
3. 控制台选择执行阶段：Part1 / Part2 / Part3 / 全部运行
